# SmartDartCorrector

### Agent Behavior
- **Movement**: Generates displacement changes (dx, dy) at each time step
- **Objective**: Minimize time and movement to hit all targets H times
- **Learning**: Uses REINFORCE algorithm to improve performance over episodes
